# Plan 02: AWS CDK Infrastructure Stacks

## Objective

Implement the AWS CDK infrastructure stacks in the `infra` package. Define the DynamoDB tables (audit log, circuit breaker state), Lambda function constructs, IAM roles, and supporting AWS resources needed by the Meridian system.

## Context

- @packages/core/src/types/index.ts — AuditLogEntry, CircuitBreakerState, Ticket, LLMResponse types
- @infra/src/index.ts — Stub to be replaced with real CDK stacks
- @infra/package.json — Current infra package (needs aws-cdk-lib added)
